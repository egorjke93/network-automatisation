# Network Configuration Backup Script

Скрипт для автоматического снятия конфигураций с сетевых устройств различных производителей.

## Описание

Данный скрипт позволяет автоматизировать процесс резервного копирования конфигураций сетевого оборудования с поддержкой устройств:
- Cisco IOS
- Juniper JunOS
- Arista EOS
- Eltex и другие

## Функциональные возможности

- Подключение к устройствам по SSH с использованием Netmiko
- Автоматическое определение типа команды для снятия конфигурации
- Сохранение конфигураций в отдельную папку с датированием файлов
- Обработка enable-режима (при необходимости)
- Логирование процесса работы
- Поддержка таймаутов для стабильной работы с медленными устройствами
